## 🌟 주요기능
1. 첫 메인 홈페이지가 달력으로써, 학생들이 놓치지 쉬운 모든 IT 관련 자료들을 한 눈에 확인할 수 있습니다. 이 중에서 스크랩한 것들만 따로 모아볼 수도 있습니다.
2. 기업소개 / IT자격증 / 지원사업 / 채용공고 탭에서 정보들을 이용 가능합니다.
3. 커뮤니티 기능으로 모각코나 자격증 스터디 모임도 구할 수 있습니다.
